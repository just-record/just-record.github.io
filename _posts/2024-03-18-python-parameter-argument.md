Parameter와 Argument의 차이

## Parameter(매개변수)

함수에 전달되는 값을 받아들이는 변수

```python
def add(a, b):
    return a + b
```

## Argument(인자)

함수를 호출할 때 전달되는 값

```python
result = add(3, 5)
```
